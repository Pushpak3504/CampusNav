How CampusNav Works

Fragment 1 – College Info

Its primary purpose is to provide users—especially new students, visitors, and faculty—with essential information about the college. This includes recent events, announcements, and general details that help users stay informed and engaged with campus activities. By placing this content upfront, we ensure that users not only benefit from navigational features but also gain a sense of connection with the college environment. This fragment acts as a dynamic bulletin board, keeping users updated while creating a welcoming and informative entry point into the app.

Fragment 2 – Q&A Bot

was created to simplify the process of finding specific locations on campus through natural text-based queries. Instead of searching manually, users can simply type questions like “Where is the Electronics and Telecommunication Department?” into an input field. This approach makes the app more interactive and user-friendly, especially for those who are new to the campus. On the backend, we’ve implemented a HashMap data structure to identify and match keywords from the user’s query. For example, if the question contains “Electronics” and “Telecommunication,” the system maps these keywords to a predefined answer stored in the HashMap. Once a match is found, the corresponding location details are displayed directly below the input box. This feature is particularly useful for quick, intuitive access to location-based information without the need for manual browsing, making navigation more efficient and accessible for all users.

Fragment 3 – AR Navigation

Fragment 3 is developed to provide an advanced and interactive navigation experience using Augmented Reality (AR). It allows users to search for various campus locations such as departments, hostels, library, and more through a dedicated search bar or by scrolling through a RecyclerView. Once a user selects a desired location, the app transitions to an integrated Unity-based module that uses the Immersal SDK for indoor AR navigation. Unlike traditional navigation apps like Google Maps, which are limited indoors, this system displays a virtual pointer guiding users step-by-step to their destination within the campus. This fragment plays a crucial role in offering precise and immersive indoor navigation, making it especially useful in large campuses where directions can often be confusing.
